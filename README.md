# Sistema de Loja

Sistema simples de controle de compras desenvolvido em Python.

## Objetivo

Simular o registro de produtos em uma compra, calculando estatísticas e informações relevantes ao final do processo.

## Funcionalidades

- Cadastro de produtos
- Soma total da compra
- Contagem de produtos acima de determinado valor
- Identificação do produto mais barato
- Encerramento controlado do sistema

## Conceitos Aplicados

- Estruturas de repetição
- Estruturas condicionais
- Acumuladores
- Comparação de valores
- Controle de fluxo

## Tecnologias

- Python 3
